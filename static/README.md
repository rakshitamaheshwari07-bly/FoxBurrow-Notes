# 🦊 FoxBurrow Notes

A cozy woodland-themed note-taking application built with Flask.

FoxBurrow Notes helps users create, organize, search, edit, and manage personal notes inside a simple and beautiful interface inspired by a forest burrow.

---

## ✨ Features

* 📝 Create, edit, and delete notes
* 🔍 Search notes instantly
* 📂 Organize notes by categories

  * Study
  * Work
  * Personal
  * Ideas
* 🌙 Dark Mode support
* 🦊 Custom woodland-themed UI
* 🍄 Animated forest-inspired design
* 🗑️ Custom delete confirmation modal
* 📱 Installable as a Progressive Web App (PWA)
* 👤 User authentication system
* 💾 SQLite database storage
